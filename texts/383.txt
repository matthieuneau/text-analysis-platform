what you will learn in this maya tutorial we'll learn how to begin maya's powerful ocean features. using maya oceans, we'll be able to achieve quality results within a matter of minutes. we'll begin this project by learning the basics of creating a maya ocean and what it is actually made of. we'll then learn about attributes we need to set early in the project. then we'll learn how to control the noisiness of waves and their length. after that, we'll create line graphs to change each wave's height, turbulence and peak. we'll finish the tutorial by adding foam and editing the color information of the shader.
